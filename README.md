Crave - Smart Recipe Application

Crave is a modern, responsive, and feature-rich recipe application built with React and Tailwind CSS. It helps users discover delicious meals, manage their favorites, and provides an immersive cooking experience with a built-in "Cook Mode" and basic AI Chef Assistant.

Follow these steps to run the project locally on your machine.

Prerequisites

Node.js (v16 or higher)

npm (Node Package Manager)

Installation

Clone the repository:

git clone [https://github.com/asadjn99/crave-recipe-app.git](https://github.com/asadjn99/crave-recipe-app.git)
cd crave-recipe-app


Install dependencies:

npm install


Install Tailwind CSS (if not already configured):

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p


Run the development server:

npm run dev


Open your browser and navigate to http://localhost:5173.
